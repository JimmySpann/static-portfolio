    BMI Chart - Capstone Project

    This web application was created in Angular 5 for the senior project of my degree. The BMI Chart was requested by JuliSu DiMucci-Ward who wants to prevent illnesses by checking for fluctuations within the body mass index of children and infants. The app records the weight and height of infants, calculates the BMI, then places data within the chart.

    The app uses RestfulAPI to send data to the front end. This allows for flexible updates and a hefty increase to security.

    Client Search

    Search for clients created on the database.

    Default Look

    This is what you see when you first log on.

    BMI Chart

    When a client is chosen, a chart will show the past data entries.

    Adding New Entry

    You can add the weight and height of a baby and it will calcuate the bmi. The calculator supports lb, kg, in, and cm.

    Edit Client Data

    If you made a mistake, you can go back and edit the records you've added.
